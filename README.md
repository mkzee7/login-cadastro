# Login e Cadastro

Projeto desenvolvido para praticar conceitos de HTML, CSS e JavaScript, com foco na criação de uma interface simples de autenticação.

## Funcionalidades

- Tela de login e cadastro na mesma página
- Alternância entre os formulários sem recarregar a página
- Mostrar e ocultar senha
- Validação dos campos de cadastro
- Validação de e-mail utilizando expressão regular (Regex)
- Mensagens de erro exibidas abaixo dos campos
- Interface desenvolvida com HTML e CSS

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento web, como manipulação do DOM, eventos, validação de formulários e organização do código em arquivos separados.

## Como executar

1. Clone este repositório.
2. Abra o arquivo `site.html` em um navegador.

## Observações

Este projeto foi desenvolvido para fins de estudo. Futuramente, ele poderá receber melhorias, como responsividade, armazenamento de usuários e integração com um backend.
